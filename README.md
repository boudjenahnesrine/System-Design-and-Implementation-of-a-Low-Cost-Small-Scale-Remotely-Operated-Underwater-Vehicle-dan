# Repository Overview

This repository contains all technical files and resources related to the design, assembly, and experimental validation of a small-scale underwater vehicle.  
It includes CAD models (`.sldprt` and `.stl`), PCB layouts, firmware, and demonstration videos.

---

## 3D Models

The mechanical structure is divided into two main components:

- **Upper hull:** sealing cover and structural enclosure  
- **Lower hull:** base structure including motor supports  

Files provided:

- SolidWorks source files (`.sldprt`)  
- STL files for 3D printing

---

## PCB Files

Includes:

- Schematics of the electronic circuits  
- PCB layouts for the control system

---

## Firmware

Includes:

- Microcontroller source code  
- Graphical user interface (GUI) code for monitoring and control
